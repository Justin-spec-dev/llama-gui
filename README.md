# llama-gui

A Windows GUI launcher for `llama-server` (from llama.cpp).

## Status

🚧 Under development. See `docs/superpowers/specs/2026-06-14-llama-gui-design.md` for the design.

## Quick start (development)

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -e ".[dev]"
python -m llama_app
```
